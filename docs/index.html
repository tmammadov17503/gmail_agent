Email Agent

Free AI email dashboard — categorizes your Gmail inbox across up to 3 accounts. Powered by Gemini Flash (free) and Gmail API. Your API keys are saved in your browser and never leave your device.

1
Get a free Gemini API key at aistudio.google.com/apikey — sign in with Google → Create API key. No billing required.
2
Create a Google OAuth Client ID at console.cloud.google.com: enable Gmail API → Credentials → OAuth 2.0 Client ID → Web app → add this page's URL as redirect URI.
3
Paste your keys below. They'll be saved in your browser — you'll never need to enter them again.

Keys are stored in localStorage in your browser only — never sent to any server.
